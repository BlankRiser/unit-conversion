The **weight** category provides conversions between metric units (grams, kilograms, tonnes) and Imperial/US units (ounces, pounds, stones, tons).

## Supported Units

- Metric: milligram `mg`, gram `g`, kilogram `kg`, tonne `t`
- Imperial / US: ounce `oz`, pound `lb`, stone `st`, ton `US ton`

## Quick Example

```ts
import { Conversion } from "@devhaven/unit-conversion";

const convert = new Conversion();

convert(1000).from("gram").to("kilogram");    
convert(1).from("pound").to("kilogram");      
```

## Common Conversions

### Gram ↔ Kilogram

```ts
convert(1000).from("gram").to("kilogram");      
convert(1).from("kilogram").to("gram");         

```
### Ounce ↔ Pound

```ts
convert(16).from("ounce").to("pound");      
convert(1).from("pound").to("ounce");       
```
