The **time** category provides conversions between milliseconds, seconds, minutes, hours, and days.

## Supported Units

- millisecond `ms`
- second `s`
- minute `min`
- hour `h`
- day `d`

## Quick Example

```ts
import { Conversion } from "@devhaven/unit-conversion";

const convert = new Conversion();

convert(60000).from("millisecond").to("minute");      
convert(2).from("hour").to("minute");                 
```

## Common Conversions

### Day ↔ Hour

```ts
convert(1).from("day").to("hour");        
convert(24).from("hour").to("day");       
```

### Second ↔ Hour

```ts
convert(3600).from("second").to("hour");      
convert(1).from("hour").to("second");         
```

### Millisecond ↔ Second

```ts
convert(1000).from("millisecond").to("second");      
convert(1).from("second").to("millisecond");         
```
