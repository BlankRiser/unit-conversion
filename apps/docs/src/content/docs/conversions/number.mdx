The **Number** category supports conversions between number bases such as binary, octal, decimal, and hexadecimal.

## Supported Bases

- binary `base 2`
- octal `base 8`
- decimal `base 10`
- hexadecimal `base 16`


## Quick Example

```ts
import { Conversion } from "@devhaven/unit-conversion";

const convert = new Conversion();

convert(255).from("decimal").to("hexadecimal")
convert(1010).from("binary").to("decimal")
```

## Common Conversions

### Binary ↔ Decimal
```ts
convert(10).from("decimal").to("binary");      
convert(10).from("binary").to("decimal");      
```

### Decimal ↔ Hexadecimal
```ts
convert("a").from("hexadecimal").to("decimal");     
convert(15).from("decimal").to("hexadecimal");      
```

### Octal ↔ Decimal
```ts
convert(77).from("octal").to("decimal");      
convert(77).from("decimal").to("octal");      
```
