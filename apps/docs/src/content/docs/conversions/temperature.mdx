The Temperature category supports conversions between Celsius, Fahrenheit, and Kelvin.

## Supported Units

- celsius `°C`
- fahrenheit `°F`
- kelvin `K`

## Quick Example

```ts
import { Conversion } from "@devhaven/unit-conversion";

const convert = new Conversion();

convert(0).from("celsius").to("fahrenheit");      
convert(212).from("fahrenheit").to("celsius");    
```

## Common Conversions


### Celsius ↔ Kelvin
```ts
convert(100).from("celsius").to("kelvin");    
convert(0).from("kelvin").to("celsius");      
```

### Celsius ↔ Fahrenheit
```ts
convert(100).from("celsius").to("fahrenheit");     
convert(32).from("fahrenheit").to("celsius");      
```
