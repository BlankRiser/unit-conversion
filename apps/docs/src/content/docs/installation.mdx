# Installation

You can install `@devhaven/unit-conversion` from [npm](https://www.npmjs.com/package/@devhaven/unit-conversion):


<PackageManagers
  pkg="@devhaven/unit-conversion"
  pkgManagers={["npm", "yarn", "pnpm", "bun"]}
/>


or [JSR](https://jsr.io/@devhaven/unit-conversion):

```bash
npx jsr add @devhaven/unit-conversion
# or
yarn add jsr:@devhaven/unit-conversion
# or
pnpm i jsr:@devhaven/unit-conversion
# or
bunx jsr add @devhaven/unit-conversion
# or
deno add jsr:@devhaven/unit-conversion
```
